# Computational-intelligence

记录计算智能优化算法的学习笔记，通过阅读论文并复现的形式加深对相关的启发式智能优化的理解。

论文复现目前有```matlab```及```python```版，由于个人能力有限，因此难以避免论文代码复现中存在的错误，如有相关不妥之处，还望指正。

在每篇的笔记当中，存在部分个人的观点，该观点仅供参考！
